To enable PWA actions, please follow these steps:

  1. Launch Microsoft Edge Browser
  2. Navigate to edge://flags
  3. Search for "App Actions"
  4. Find the entry "Enable App Actions on Windows for web apps" (Flag name: #edge-app-actions-on-windows-for-web-apps)
  5. Change the status to "Enabled"
  6. Click the "Restart" button in the lower right
